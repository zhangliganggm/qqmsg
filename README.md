QQMsg 
===================================  
  发送文字或者图片消息给QQ聊天窗口.
  
    
使用说明  
-----------------------------------  
  安装了QQ,并且用官网安装包安装的。不能是绿色版。如需确认。直接查看注册表看腾讯键值是否存在。
  目前只支持windows.后续如有需要会开发 基于webqq 协议来发送消息.

### 参数说明  
  调用 qqmsg.py 
  参数1 qq_number
  参数2 message

### 注意你的cmd编码。中文windows是GBK。其他的没测试.  
  
### 完整范例  
    qqmsg.py 394452216 hello world
